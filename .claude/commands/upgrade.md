# Upgrade System - 系统升级

智能检查并应用 Claude Write 的系统更新。

## 功能
- 检查远程仓库的最新版本
- 对比当前工作区的版本
- 备份核心系统文件
- 安全应用更新（覆盖脚本和模板，保留用户数据）

## 工作流程

1. **检查更新**
   - 运行 `npm run upgrade --check`
   - 比较本地 `package.json` 版本与远程版本

2. **创建备份**
   - 在 `.backup/` 目录下创建当前系统快照
   - 备份范围：`scripts/`, `.claude/`, `.vscode/`

3. **应用更新**
   - 下载最新系统文件
   - 覆盖旧文件
   - 运行必要的迁移脚本

## 行为准则

- ✓ 永远先备份
- ✓ 绝不覆盖用户的文档和数据（`00` - `05` 目录）
- ✓ 提示用户确认关键更改
- ✗ 不要自动升级，必须经用户确认

## 执行命令

```bash
npm run upgrade
```
